# Conventions & stacks par défaut (floSa)

## Profil

Ingénieur Data / ML / AI. Domaines : data science, data engineering, MLOps, ML engineering, AI engineering. Plutôt solo, parfois petite équipe. Spécialité **on-prem** (industriels, ESN).

## Posture

- Outils opinionated et productifs : `uv` plutôt que `pip + venv`, `ruff` plutôt que `flake8 + isort + black`.
- Clarté avant cleverness.

## Stacks par défaut

| Besoin | Choix par défaut |
|--------|------------------|
| Langage | Python (géré par `uv`) |
| API web | FastAPI |
| Manipulation de données | pandas, polars |
| ML classique | scikit-learn |
| Deep learning | PyTorch |
| App data interactive | Streamlit |
| Packaging | Docker |
| CI | GitHub Actions |
| Lint / format | ruff |
| Connaissance | Obsidian + Claude Code + DevBrain |

## Voix des pages

- Français, phrases courtes, pas de marketing-speak.
- Ton impersonnel dans les fiches (ni « tu » ni « vous »).
- Pas d'émojis sauf si le contexte en utilise.
