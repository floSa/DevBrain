# Archétypes de projet

Sept profils types. Le skill planifier-projet identifie d'abord l'archétype, puis branche les seules questions pertinentes (cf. [[questions-projet]]).

1. **Analyse / exploration de données** — notebook, stats, viz, rapport.
2. **App data/ML interactive** — Streamlit : annotation, dashboard temps réel.
3. **ML/IA algorithmique** — jeux (MCTS/CFR), détection image, reco.
4. **Pipeline data / ingestion** — ELT, robustesse, ops.
5. **RAG / app LLM** — retrieval, LLM, knowledge graph, éval.
6. **Tuto / apprentissage** — repro, pas de prod.
7. **Réplique ludique** — perso.

## Axe transverse : on-premise

Quel que soit l'archétype, trancher tôt : cloud autorisé ou on-prem strict / air-gapped ? GPU ou CPU only ? Données sortant du réseau client ? (détail dans [[questions-projet]]).
