# Frameworks

<!-- AUTO:START -->
Briques techniques de la catégorie `framework/*`.

- [[Dev/Services/FastAPI|FastAPI]] — Framework web Python asynchrone : API typées sur Starlette + Pydantic, doc OpenAPI générée automatiquement.
- [[Dev/Services/Flask|Flask]] — Micro-framework web Python (WSGI) minimaliste et extensible : noyau réduit (routage Werkzeug + templates Jinja2), tout le reste ajouté à la carte par extensions.
- [[Dev/Services/HTMX|HTMX]] — Bibliothèque hypermedia : des attributs HTML déclenchent des requêtes AJAX et remplacent des fragments de page renvoyés en HTML, pour de l'interactivité riche sans JavaScript lourd.
- [[Dev/Services/Jinja2|Jinja2]] — Moteur de templates Python rapide et expressif : gabarits HTML avec héritage, échappement automatique et expressions proches de Python ; le moteur de templates de Flask.
- [[Dev/Services/Prisma|Prisma]] — ORM TypeScript nouvelle génération : schéma déclaratif, client typé et migrations générées.
- [[Dev/Services/SQLAlchemy|SQLAlchemy]] — Toolkit SQL et ORM Python de référence : couche Core d'expression SQL + ORM Data Mapper, entièrement typé depuis la 2.0.
- [[Dev/Services/SQLModel|SQLModel]] — Une couche fine au-dessus de Pydantic et SQLAlchemy : une seule classe typée sert à la fois de modèle de validation et de table ORM, taillée pour FastAPI.
- [[Dev/Services/Uvicorn|Uvicorn]] — Serveur ASGI Python performant (uvloop/httptools) qui exécute les applications async comme FastAPI.
